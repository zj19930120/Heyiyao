# 核医疗产业综合信息平台

一个专业的核医疗产业信息展示平台,集成了元素周期表、产业图谱、数据大屏等多个功能模块。

## 🌟 功能特性

### 1. 核医疗产业综合信息平台 (index.html)
- 📊 产业链全景展示
- 🔬 核医学基础知识
- 📈 产业图谱可视化
- 🎯 交互式信息导航

### 2. 核医学元素周期表 (yuansuzhouqibiao.html)
- 🧪 完整的元素周期表展示
- 💊 关键核素标注
- 📝 详细的元素信息
- 🎨 直观的可视化设计

### 3. 核医药产业数据大屏 (dashboard.html)
- 📊 实时数据展示
- 📈 产业链分析
- 🌐 全球市场概览
- 💡 动态粒子背景效果

### 4. 放射性药物原理 (放射性药物原理.html)
- 🔬 药物作用原理
- 📚 专业知识展示
- 🎓 教育培训资源

## 🚀 快速开始

### 在线访问
直接打开任意HTML文件即可在浏览器中查看:
- `index.html` - 主入口页面
- `yuansuzhouqibiao.html` - 元素周期表
- `dashboard.html` - 数据大屏
- `放射性药物原理.html` - 药物原理

### 本地部署
1. 克隆仓库到本地
```bash
git clone https://github.com/你的用户名/你的仓库名.git
```

2. 使用浏览器打开HTML文件
```bash
# 直接双击HTML文件,或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **可视化**: 原生Canvas, SVG
- **特效**: Particles.js (粒子背景)
- **工具**: Python (文档处理)

## 📖 使用说明

### 元素周期表
- 点击元素查看详细信息
- 不同颜色标注不同类型的核素
- 支持搜索和筛选功能

### 数据大屏
- 自动轮播展示关键数据
- 响应式布局适配不同屏幕
- 动态粒子背景效果

### 产业图谱
- 查看核医疗产业链全景
- 了解各环节关键企业
- 掌握产业发展趋势
